    ## About Me

    👋 大家好，我是 HEBUST 的一名在读学生，生活和学习在充满活力的河北科技大学。欢迎来到我的个人小窝，这里记录着我的思考、观察与热爱。很高兴能在这里与你相遇，并分享我的世界。

    ### 📸 点滴日常

    我是一名普通的在校大学生，正经历着人生中一段充满探索和发现的时光。在大学里，我不仅学习专业知识，更在学习如何与自己、与世界更好地相处。

    - **定格时光的摄影爱好者**：对我而言，摄影远不止是按下快门，它是我与世界对话的方式，是一场关于光影、构图和情感的奇妙旅程。我热衷于用镜头捕捉生活中的瞬间——可能是校园里的一道斜阳，旅途中的一瞥微笑，或只是发呆时窗台上的一缕光。这些被定格的画面，是我对生活热爱的证明。我相信，精妙的构图和恰当的光线能为照片注入灵魂。

    - **故事与声音的沉迷者**：我享受沉浸在不同故事里的感觉。无论是 看剧 、 听播客 还是 看纪录片 ，我都乐于从中感知更广阔的世界和更多元的人生。这些媒介让我得以窥见他人的思想与生活，是我放松和汲取灵感的重要方式。

    - **灵魂的漫游者**：身体和灵魂，总要有一个在路上。 旅行 让我走出日常的圈子，去体验不同的风土人情；而 发呆 和 晒太阳 则是我给心灵的“小憩”，在这些看似“无用”的时刻里，我常常能收获内心的平静与新的灵感。

    - **布鲁斯（Blues）的聆听者**：我尤其喜欢听 布鲁斯音乐 。它被公认为众多现代流行音乐的源头之一，以其独特的律动和真挚的情感表达深深吸引着我。这种音乐风格里蕴含的质朴力量和故事感，常常能让我产生共鸣，为我带来慰藉和力量。


    ### 🌟 期待相遇

    感谢你阅读到这里。在这个角落里，我会持续分享我的摄影作品、观影听歌笔记、旅行见闻以及那些平凡日子里的思考碎片。

    如果你也对生活中的这些小美好充满好奇，欢迎常来看看，或者通过评论与我交流。希望我们都能在这里找到一片属于自己的精神自留地。
